MercuryLoyaltyTM Easy Start Guide

MercuryLoyalty is a full-featured loyalty program. Mercury’s gift card program provides an additional value that developers and their resellers can leverage to sell more POS systems and to retain their merchant base.

Integration is simple!

 

First subscribe to our EndPoint:

Starting your integration is simple; post data to our RESTful endpoint with the test credentials. http://api.cert.mercuryloyalty.com/v3_0/loyalty/ A SOAP endpoint is available too!

 

Second, use the basic Transactions:

There are two basic transactions to get Loyalty up and running! GetStatus or AddCredits.

Transaction

HTTP

Description

GetStatus

GET

Issue is the first transaction performed on a new gift card. It activates the account and loads a specified amount balance onto the card.

AddCredits

GET

Takes value off of the card, reducing the available balance by that purchase amount. Declines the transaction in case of insufficient balance.

Examples:

http://api.cert.mercuryloyalty.com/v3_0/loyalty/GetStatus/?apikey=PAUL&apisecret=f2ee0e69a14c45039c6041b19fd18d88&customeridentifier=4076374996
